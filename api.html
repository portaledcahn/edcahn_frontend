Proveedores Retorna el listado de proveedores
GET
/api/top/supplier
Retorna el listado de proveedores con más contratos o más pagos
Parametro Valor Descripción Tipo de Parametro Tipo de Dato
section Honducompras variables de tipo texto para determinar si se necesitan el top de proveedores por contratos o topo de proveedores por pagos enviando los valores de “contract” y “implementation” respectivamente. query string
par Oncae variable de tipo texto para determinar si se necesita el top de proveedor por cantidad de contratos o montos de contrato enviando los valores “amount”, “count” respectivamente query string
